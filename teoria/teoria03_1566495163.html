1

Опишите теги, которые записываются в контейнере HEAD и поддерживаются в HTML5
Ответ:Тег head предназначен для хранения других элементов, цель которых помочь браузеру в работе с данными.
Также внутри head находятся теги, которые используются для хранения информации предназначенной для браузеров и поисковых систем.
Например, механизмы поисковых систем обращаются к тегам для получения описания сайта, ключевых слов и других данных.
Содержимое тега head не отображается напрямую на веб-странице, за исключением тега title устанавливающего заголовок окна веб-страницы.
Внутри контейнера head допускается размещать следующие элементы: base, basefont, bgsound, link, meta, script, style, title.

2

Что такое "Семантические теги"? Почему их полезно использовать?
Ответ:Семантические теги — это теги, которые предназначены для того чтобы компьютерные программы поисковые системы и браузеры
понимали какой тип информации заложен в данных тегах.
Политика спецификации HTML5 заключается в том, чтобы каждый HTML-тег имел свою семантику.
Симантические теги полезно изпользовать потому что они позволяют создать основу ну или скелет веб-страницы.

3

Что такое CSS-селектор? Какие они бывают?
Ответ:Селектор в CSS — это указатель на элемент или группу элементов, к которым будут применяется правила стилей.

    Существуют следующие виды селекторов:
      1. Селекторы по тэгам
        2. Селектор по id
          3. Селекторы по классам
            4. Селектор по нескольким классам
